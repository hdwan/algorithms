# 前缀和

## 解析

数组：$nums[0], nums[1, nums[2], ..., nums[n - 1]$。

前缀和：$preSum[i + 1] = s[i] + nums[i]$，$preSum[0]= 0$。

$preSum$ 的长度为 $n+1$。

子数组 $[*left*,*right*)$ 的元素和为 $preSum[right] − preSum[left]$。
