# 单调栈

## 知识总结

栈内元素**有序**，递增或者递减。

**常见模型**：找出每个数左（右）边离它最近的比它大/小的数  check处检测大小

```java
// 使用数组模拟栈
int[] stk = new int[n];
int pre = 0; // 指向栈顶元素上一个位置，等价于栈中元素个数。
```

**单调栈**

```c++
int pre = 0; 
for (int i = 1; i <= n; i ++ )
{
    while (pre != 0 && check(stk[pre - 1], i)) pre -- ;
    stk[pre++] = i;  // 当前元素无条件加入
}
```
